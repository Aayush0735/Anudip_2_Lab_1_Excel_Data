# Sales Data Analysis (Excel / Google Sheets)

## Overview
This project involves analyzing sales data using Excel/Google Sheets, including data cleaning, transformation, and visualization through pivot tables and charts.

## Dataset
The dataset contains:
- Sales data across multiple months
- Region-wise performance
- Category-wise sales distribution
- Revenue and transaction details

## Key Tasks Performed

### 1. Data Cleaning
- Removed inconsistencies and formatting issues
- Standardized currency and numeric values
- Handled missing or incorrect data entries

### 2. Data Analysis (Pivot Tables)
- Created pivot tables to analyze:
  - Region-wise Net Sales
  - Category-wise performance
  - Monthly sales trends

### 3. Data Visualization
Generated the following charts:
- **Column Chart** → Region vs Net Sales  
- **Pie Chart** → Category Sales Distribution  
- **Line Chart** → Month-wise Sales Trend  

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Visualization Tools

## Insights
- Identified top-performing regions and categories
- Observed monthly sales growth trends
- Highlighted key revenue-driving segments

## Outcome
- Improved understanding of sales patterns
- Enabled data-driven decision-making through visual insights
